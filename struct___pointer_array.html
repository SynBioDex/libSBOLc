Detailed Description

A dynamic array of void pointers.

Used anywhere a variable number of SBOLObjects needs to be stored. It would be called "SBOLObjectArray", except that descendants of SBOLObject start with a pointer to their base struct rather than the struct itself, so you can only cast part of them to (SBOLObject *). That's probably fixable though.
